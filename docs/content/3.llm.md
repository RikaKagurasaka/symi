本页说明如何在当前文档站中使用已经配置好的LLM支持

## LLM支持
你可以复制以下两个链接或之中的内容到支持文本输入的LLM工具中（如ChatGPT、Claude等），以便在对话中引用文档：
- [llms.txt](/llms.txt){target="_blank"}：精简、结构化的文档索引，适合让模型先建立目录感知。
- [llms-full.txt](/llms-full.txt){target="_blank"}：更完整的文档文本，适合上下文充足时直接检索。

### MCP 工具说明

本站提供了两个工具：

- `list-pages`：列出可用文档页面（标题、路径、描述）。
- `get-page`：根据页面路径读取完整文档内容。

推荐调用流程：

1. 先调用 `list-pages` 找到目标主题的路径。
2. 再调用 `get-page` 拉取对应页面全文。
